## Structure
```
- dicoding-android-submission
    - buildSrc (Kotlin Library)
        - dependencies
    - app (Android Application)
        - UI / Main
    - data (Android Library)
        - Data Source
    - domain (Android Library)
        - Model
    - common (Android Library)
        - Shared Resource
    - favorite (Android Dynamic Feature)
        - Favorite Feature
```
